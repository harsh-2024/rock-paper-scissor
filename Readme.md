# WELCOME TO ROCK-PAPER-SCISSOR-GAME
* This is the simple game I made using python.
* There are three choices that you can make by pressing
   0 -> for the ROCK
   1 -> for the PAPER
   2 -> for the SCISSOR
* On the other hand, your opponent is the computer.
